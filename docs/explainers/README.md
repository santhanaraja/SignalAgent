# docs/explainers/ — the teaching layer

Concept and event explainers: prose with embedded mermaid, cross-linked from
the records they illuminate. The genre boundary in one line — **briefs are
deliberations** (the four-questions documents a ruling weighs),
**decisions are rulings** (the registry's permanent record of what was chosen
and why), **explainers are the teaching/exhibit layer**: they take something
the system already did or already encodes and make it legible — a live event
walked through its mechanics, a design rule taught from first principles, a
methodology named. Explainers never rule anything; every number in one must
match the committed close-basis record it narrates.
